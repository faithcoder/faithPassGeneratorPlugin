== Description ==

The Password Generator plugin allows you to easily generate random passwords on your WordPress site. Users can insert the provided shortcode into a page or post to display the password generator box.

= Features =

- Simple and easy-to-use password generator.
- No need to save changes, just insert the shortcode and start generating passwords.

== Installation ==

1. Upload the `password-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the 'Password Generator' settings page under 'Settings' in the WordPress admin to get the shortcode.

== Usage ==

To display the password generator, insert the following shortcode into any page or post:

[password_generator]


No need to save changes after inserting the shortcode. Users can customize the password length directly in the generator box.
